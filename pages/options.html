How it works:
  • Open a site and toggle Unlock Copy or Force Mode.
  • That site is added here with the chosen mode.
  • Remove it from this list to stop applying it.
Notes:
  • Unlocking copy can break some site interactions.
  • Some sites use overlays to block selection; you may need to remove them manually.
  • You can remove this extension at any time.
  • Built for Google Chrome.
Diagnostics:
  • Each entry shows the last enabled time and mode.
Limitations:
  • Some sites cannot be unlocked due to strict security policies.
  • Iframe-heavy sites may still resist, even with Force Mode.
Privacy:
  • No data is collected or transmitted.
  • Site preferences are stored locally in your browser.
Permissions:
  • tabs and activeTab: detect the active tab and reload when toggles change.
  • storage: save your per-site preferences.
  • scripting: inject enable scripts on sites you choose.
  • Host permissions: allow running on any site you opt into.
